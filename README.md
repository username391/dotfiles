## Install Stow
```
sudo pacman -S stow
```

## Download Repo
```
git clone https://github.com/username391/dotfiles.git
cd dotfiles
```

## Create Symlinks
```
stow .
```


## Tmux
```
sudo pacman -S tmux
```

Download tmux package manager
```
git clone https://github.com/tmux-plugins/tpm ~/.tmux/plugins/tpm --depth=1
```

Install tpm with `<PREFIX> + I`
